This project is a simple Henna Design Gallery website created using HTML, CSS, and JavaScript. 
The website showcases different henna designs in a clean and visually appealing layout.

Features:
- Image gallery to display henna designs
- Clickable images with modal preview
- Feedback section for user comments
- Responsive and simple UI

Purpose:
This project was developed to practice front-end web development concepts such as layout design, DOM manipulation, and user interaction.

Technologies Used:
- HTML
- CSS
- JavaScript
